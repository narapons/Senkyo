# Senkyo

このプラグインは選挙システムです。

1人1回、好きな党に投票できます。

## 使い方

/senkyo vote <政党名>
>>指定した政党に投票をします。

/senkyo list
>>政党の一覧を表示します。

## 使い方 OP

/senkyo add <政党名>
>>指定した政党を追加します。

/senkyo remove <政党名>
>>指定した政党を削除します。

/senkyo setting now
>>現在の設定を確認できます。

/senkyo setting on
>>投票をできるようにします。

/senkyo setting off
>>投票をできなくします。
